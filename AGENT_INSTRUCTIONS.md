# 喵姆 AI 七維分析引擎 — Cowork Agent 任務指令書

## 你是誰
你是喵姆（MiaoMu）AI 股市偵測站的系統建構 Agent。你的任務是根據本套件中的 SKILL.md（分析框架）和 scripts/miaomu_engine_v2.py（Python 引擎），完整建構一套「讓普通投資人像開外掛一樣看穿股市假象」的自動化分析系統。

## 系統願景
在喵姆面前，所有市場數據都是透明的。外資的假動作、財報的紙上富貴、管理層語氣的微妙變化——全部無所遁形。喵姆不是告訴你「該買什麼」，而是告訴你「誰在騙你、誰在洗盤、誰在出貨」。

## 你要做的事（按順序）

### Phase 1：理解框架
1. 讀完 SKILL.md，理解七維分析架構（四維基礎 + 三維深度）
2. 讀完 scripts/miaomu_engine_v2.py，理解每個模組的輸入/輸出格式
3. 確認你理解以下核心邏輯：
   - 三方共振判讀表（籌碼維度的靈魂）
   - 一票否決機制（財報異常 / 出貨陷阱 / 敘事崩塌）
   - 惡魔代言人會攻擊自己的結論
   - 眉角提問武器庫（Tier 1-4）可以發給不同 AI

### Phase 2：建構資料接口
喵姆需要從以下免費來源自動抓取數據，填入七維引擎：

**台股免費 API（必接）：**

| 資料 | 來源 | 用途 | 填入維度 |
|------|------|------|---------|
| 即時/收盤股價 | TWSE OpenAPI `openapi.twse.com.tw` | 技術指標計算 | 維度三 |
| 三大法人買賣超 | TWSE 每日收盤資料 | 外資/投信淨買 | 維度二 |
| 融資融券 | TWSE 信用交易統計 | 融資變化 | 維度二 |
| 月營收 | 公開資訊觀測站 MOPS | 營收趨勢 | 維度一 |
| 季財報 | MOPS 財務報告 | OCF/淨利/應收/存貨 | 維度一 |
| 董監事持股異動 | MOPS 內部人交易 | 內部人信號 | 維度二+七 |
| 歷史日K線 | Yahoo Finance (`yfinance`) | 均線/RSI/MACD | 維度三 |

**建構 data_fetcher.py：**
- 封裝所有 API 呼叫
- 自動把抓到的原始數據轉換成 miaomu_engine_v2.py 需要的 dict 格式
- 加入錯誤處理和重試機制
- 加入資料快取（避免重複請求）
- 每筆資料標注時間戳

**注意：**
- 維度四（情緒）、五（惡魔代言人）、六（產業結構）、七（先行指標）的部分資料無法從免費 API 取得
- 這些維度的補充方式是：用 Tier 1-4 提問發給外部 AI（Perplexity/Claude/Gemini）
- 建構 `prompt_dispatcher.py` 負責自動產出提問並解析回傳結果

### Phase 3：建構計算引擎
基於 miaomu_engine_v2.py，補完以下技術指標的即時計算：

**technical_calculator.py：**
```python
# 需要計算的指標（從日K線原始數據）
- MA5, MA20, MA60（移動平均線）
- RSI(14)
- MACD (12, 26, 9) 及柱狀體
- 布林通道 (20, 2)
- 20日均線斜率（角度）
- RSI/MACD 背離偵測
- 成交量比（5日均量/20日均量）
```

建議使用 `pandas` + `numpy` 計算，或 `ta-lib` 如果環境允許。

### Phase 5：建構自動化排程
建立 `daily_scanner.py`——每天的自動化流程：

```

1. 抓取自選股清單的最新數據（data_fetcher）
2. 計算技術指標（technical_calculator）
3. 跑七維引擎（miaomu_engine_v2）
4. 產出異常排行榜（anomaly > 6 的推送）
5. 對異常標的自動產出 Tier 1-4 提問
6. 格式化輸出（LINE / 網頁）
7. 推送到 LINE Bot

每週排程（週五收盤後）：
1. 彙總本週所有異常事件
2. 產出「週逆勢觀點」
3. 掃描內部人交易異動（眉角速報）
```

## 關鍵設計原則

1. **普通人 30 秒讀完**：所有輸出都必須有「白話版」，不能只有數字
2. **眉角必須突出**：每份報告的「眉角」和「逆勢觀點」要用視覺上最醒目的方式呈現
3. **惡魔代言人不能省略**：每個 BUY 裁決都必須經過反向驗證，沒有例外
4. **數據來源必須標注**：每筆資料都要標時間戳和來源，讓使用者知道資料的新鮮度
5. **安全護欄**：永遠不承諾報酬率，永遠附帶免責聲明
6. **在喵姆面前都是裸體的**：這個系統的目的就是看穿一切假象——外資的、公司派的、分析師的
