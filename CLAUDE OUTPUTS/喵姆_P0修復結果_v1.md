## 修復內容

**檔案：** `scripts/miaomu_engine_v2.py`，IndustryStructure.analyze() 方法

**問題：** 當 `data={}` 時，所有 `.get()` 取到 default 值：
- `moat_type="none"` → score -1.0
- `moat_durability="weak"` → score -2.0
- 最終 score = 5.0 - 1.0 - 2.0 = **2.0**，signal = **danger**

**修法：** 在 analyze() 開頭加入空 dict 偵測，直接回傳 neutral fallback：

```python
# P0 修復：空 dict 不應產生 danger 信號，回傳中性 fallback
if not data:
    return DimensionResult(
        name="產業結構",
        score=5.0, anomaly=5.0,
        signal="neutral",
        plain_text="產業資料不足｜以中性評估",
        key_finding="無產業資料輸入｜fallback neutral",
        mekak="缺乏產業資料，無法判斷護城河與敘事風險",
        details={"fallback": True}
    )
```

|------|----------|--------|--------|
| 2330 台積電 | 上漲＋外資中度買超 | WAIT/Low | **WAIT/Low**（技術面 neutral，需 ≥3 bullish 才觸發 BUY） |
| 2382 廣達 | 強漲＋外資大買 | WAIT/Low | **WAIT/Low**（同上，但籌碼 score=9.0/bullish） |
| 2002 中鋼 | 下跌＋外資賣超＋融資增 | WAIT/Low | **SELL/High** ✅ |
| 2379 瑞昱 | 暴跌＋外資大賣＋融資暴增 | WAIT/Low | **SELL/High** ✅ |

### 關鍵差異

- **修復前：** 4/4 全部 WAIT/Low（零鑑別力）
- **修復後：** 2 種裁決（WAIT + SELL），空方信號正確觸發

### 為何多方股票未觸發 BUY？

VerdictEngineV2 的 BUY 條件需要 ≥3 個 base dimensions 為 bullish。目前 4 個 base dimensions 中：
- 財報真實性：永遠 neutral（無資料）
- 市場情緒：永遠 neutral（無資料）
- 這代表最多只有 2/4 可能 bullish（籌碼 + 技術面）

**結論：BUY 判定需要至少 P1 改善（接入財報 or 情緒資料）才能觸發。但 SELL 判定已正常運作。**
