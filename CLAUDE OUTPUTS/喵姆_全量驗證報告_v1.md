## 九、具體行動建議

### 立即可做（Phase 2 前置）

1. **`is_fallback` 升級為 engine 原生欄位**
   - 在 `miaomu_engine_v2.py` 的 DimensionResult 加 `is_fallback: bool`
   - engine 在 score=5.0 且 raw_data 含 `fallback` key 時自標記
   - report_to_stock_dict 直接讀取，不再靠字串匹配
   - 評分計算排除 fallback 維度

2. **PE 異常值處理**
   - PE=0 或 PE>100 時標記 `pe_anomaly: true`
   - 估值狀態回傳「PE 失真，不參與估值」而非硬算百分位

3. **殖利率接入**
   - FinMind 有 `TaiwanStockDividend` dataset
   - 與月營收同方式接入

### Phase 2 核心（需要較大工程量）

4. **產業結構：低配真資料版**
   - 從 FinMind 或 TWSE 取得產業分類
   - 用同產業 PE 中位數做 peer comparison
   - 從 100% fallback 降到至少有「產業定位」

5. **市場情緒：簡易版**
   - 用融資餘額變化 + 成交量異常做簡易情緒指標
   - FinMind 有 `TaiwanStockMarginPurchaseShortSale`

6. **先行指標：簡易版**
   - 用台股加權指數 + 外資期貨部位做大盤先行
   - 比完全沒資料的假中性好
