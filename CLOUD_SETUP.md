# ☁️ 喵姆雲端同步設定教學

## 你需要的東西
- Google 帳號（你已經有了）
- 5 分鐘的時間

## Step 1：建立 Google 試算表

1. 打開 Google Drive → 進入「**股票分析**」資料夾
2. 右鍵 →「新增」→「Google 試算表」
3. 把試算表命名為「**喵姆持股資料**」

## Step 2：貼上程式碼

1. 在試算表的上方選單 → 點「**擴充功能**」→「**Apps Script**」
2. 會打開一個新分頁，裡面有預設程式碼
3. **全選刪除**預設程式碼
4. 打開專案裡的 `Code.gs` 檔案 → **全選複製** → **貼上**到 Apps Script
5. 按上方的 💾（儲存）

## Step 3：初始化工作表

1. 在 Apps Script 上方，函式選單選「**initSheets**」
2. 按「▶️ 執行」
3. 第一次會跳出授權視窗：
   - 點「審查權限」
   - 選你的 Google 帳號
   - 點「進階」→「前往 喵姆持股資料（不安全）」→「允許」
4. 看到「✅ 初始化完成」的提示 → 成功！

回到試算表，應該會看到 `portfolio`、`profile`、`_meta` 三個工作表。

## Step 4：部署為 Web App

1. 在 Apps Script → 點右上角「**部署**」→「**新增部署**」
2. 左邊齒輪圖示 → 選「**網路應用程式**」
3. 設定：
   - 說明：喵姆同步 API
   - 執行身分：**我**
   - 誰可以存取：**所有人**
4. 按「**部署**」
5. **複製**那個「網路應用程式網址」（長得像 `https://script.google.com/macros/s/xxxxx/exec`）

## Step 5：在喵姆裡設定

1. 打開喵姆 → 進入管理後台
2. 點「**☁️ 雲端同步**」分頁
3. 把剛才複製的 URL 貼到輸入框
4. 按「**測試連線**」→ 看到 ✅ 就代表成功了！

## 使用方式

- **⬆️ 上傳到雲端**：把目前的持股、設定檔上傳到 Google Sheets
- **⬇️ 從雲端下載**：從 Google Sheets 下載持股到這個瀏覽器
- **🔄 自動同步**：開啟後，每次新增/修改持股會自動上傳

## 常見問題

**Q: 換了瀏覽器/電腦怎麼辦？**
A: 在新瀏覽器的喵姆後台貼上同一個 Web App URL，點「從雲端下載」就好。

**Q: 會不會被別人看到？**
A: 資料存在你自己的 Google 帳號裡。雖然 Web App 設定為「所有人可存取」（技術上必須這樣才能從靜態網站呼叫），但別人不知道你的 URL，而且 URL 本身就是一組隨機密碼。

**Q: 費用？**
A: 完全免費。Google Apps Script 的免費額度對個人使用綽綽有餘。

**Q: 更新程式碼後要重新部署嗎？**
A: 如果修改了 Code.gs，需要到 Apps Script → 部署 → 管理部署 → 編輯 → 版本選「新版本」→ 部署。
