# 喵姆地緣政治歷史情境比對引擎

## 核心理念

> 「歷史不會完全重演，但它押韻。」— Mark Twain

市場每天的新聞都不同，但人類對恐懼和貪婪的反應模式是固定的。
這個引擎的任務：把今天的事件，對應到歷史上最相似的案例，讓 AI 的判斷有歷史依據，不是憑空猜測。

---

## 執行流程

### Step 1：事件分類

收到使用者輸入後，先將事件歸類到以下類型之一：

```
A. 地緣政治衝突（戰爭、軍事行動、領土爭端）
B. 金融危機（銀行危機、貨幣崩潰、債務危機）
C. 能源衝擊（油價暴漲、供應中斷）
D. 貿易戰與制裁（關稅、出口管制、科技禁令）
E. 疫情與天災（全球性衝擊）
F. 政治動盪（政變、選舉衝擊、政策急轉）
G. 科技泡沫（估值崩潰、產業革命）
H. 通膨與升息（央行政策衝擊）
```

### Step 3：情境相似度評分

比對當前事件與歷史案例，從五個維度評分：

```
1. 衝突規模相似度（局部/區域/全球）         0–20分
2. 能源供應鏈威脅程度                        0–20分
3. 直接涉及台灣/半導體程度                   0–20分
4. 伴隨金融壓力程度（通膨/升息/信貸）        0–20分
5. 歷史解決時間相似度（快速/中期/長期）      0–20分

總分 100分
80–100：高度相似，歷史參考價值極高
60–79：中度相似，參考但需注意差異
40–59：部分相似，謹慎參考
40以下：情境較新，歷史參考有限
```

### Step 4：核心規律提取

無論何種事件，套用以下經過驗證的歷史規律：

**規律一：利空出盡定律**
```
「買謠言，賣事實」的反向版本：
開戰前恐慌最大 → 開戰後反而上漲
歷史案例：波灣戰爭1991、伊拉克戰爭2003
適用條件：戰爭結果相對可預期，不威脅全球供應鏈
```

**規律二：能源例外定律**
```
唯一讓股市長期下跌的戰爭類型：
封鎖全球能源供應鏈的衝突
1973年石油危機是最極端的例子
現代版警示：霍爾木茲海峽封鎖超過3週
```

**規律三：時間修復定律**
```
超過50個地緣政治事件的統計：
- 1個月後：平均回升60%的跌幅
- 6個月後：平均轉正
- 1年後：平均漲11%
例外：伴隨金融危機的複合衝擊（2001、2022）
```

**規律四：恐慌賣出代價定律**
```
在歷次衝突最恐慌時賣出的投資人：
- 平均錯過後續36%的反彈
- 回補時間平均延遲8個月
- 最終報酬落後持有者15–20%
```

**規律五：存股護城河定律**
```
高股息存股在地緣衝擊中的表現：
- 短期跌幅通常比成長股小30–50%
- 股息持續發放，提供心理錨定
- 長期（3年+）幾乎全數收復失地
台灣高股息ETF（0056、00919等）的歷史驗證：
每次重大衝擊後18個月內均回到前高
```

## 特殊情境處理

### 當事件為「複合衝擊」時（最危險）

複合衝擊 = 地緣事件 + 以下任何一項：
- 高通膨（>5%）同時發生
- 聯準會正在升息循環中
- 全球信貸市場出現壓力
- 科技股本益比超過歷史均值1.5倍以上

複合衝擊的歷史結果與單純地緣衝擊完全不同：
```
單純地緣衝擊 → 快速回復（平均6個月）
複合衝擊    → 延長下跌（平均18–24個月）

2022年是最近的典型複合衝擊：
俄烏戰爭 + 40年最高通膨 + 暴力升息
結果：S&P 500全年跌19.4%，遠超單純地緣衝擊
```

遇到複合衝擊時，輸出中必須加入警示：
```
⚡ 複合衝擊警示：此次事件同時伴隨[X]，
   歷史上複合衝擊的平均修復時間為18–24個月，
   建議適度降低倉位至防禦水位。
```

### 當「霍爾木茲封鎖」威脅出現時

這是對台股最重要的單一地緣風險因子：

```
霍爾木茲威脅等級評估：

等級1（口頭威脅）：歷史上從未真正封鎖 → 市場通常過度反應後回調
等級2（部分限制）：油價+20–40% → 台灣製造業成本上升，但可承受
等級3（實質封鎖超過2週）：油價可能破$100 → 全球通膨再起，Fed政策受限
等級4（封鎖超過1個月）：歷史無先例 → 系統性風險，考慮降低股票曝險

目前最接近的歷史案例：1973年石油禁運（最終重創市場）
但現代差異：美國已成能源淨出口國，全球衝擊程度可能低於1973
```

## 使用範例

**使用者輸入：**
「伊朗對霍爾木茲海峽的油輪發出警告，美國增派航母，台股今天跌了3%」

**引擎執行：**
1. 分類 → 類型A（地緣衝突）+ 類型C（能源威脅）複合
2. 比對 → 最相似案例：1990年波灣戰爭（85%）、1973年石油危機（40%）
3. 評分 → 能源威脅20/20、台灣涉及度12/20、金融壓力8/20、總分72分（中高度相似）
4. 規律提取 → 1990年模式：開戰後反而上漲；但需監控霍爾木茲實際封鎖情況
5. 輸出個人化建議
