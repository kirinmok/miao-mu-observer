> 版本：V37 規劃 · 日期：2026-03-07

## 一、系統願景

喵姆「明日預報」不是算命，是**把全世界的公開資訊用 AI 串起來，比對歷史上類似情境，告訴你「現在的局勢最像歷史上的哪一次，那次後來怎麼了」**。

核心邏輯：
```
即時全球事件 → AI 關聯性分析 → 歷史模式比對 → 明日情境預判 → 操作建議
```

就像氣象預報不是「保證明天下雨」，而是「根據氣壓、風向、濕度，明天下雨機率 80%」。

## 三、「明日預報」核心模組設計

### 模組 A：全球事件雷達 (Global Event Radar)

```
輸入：GDELT API → 過去 24-72 小時的事件流
處理：
  1. 篩選與台股相關的事件（關鍵字：Taiwan, semiconductor, TSMC, China, trade war, oil, Fed...）
  2. 計算衝突分數趨勢（3天移動平均 vs 7天移動平均）
  3. 標記「異常事件」（衝突分數突然跳升 > 2 標準差）
輸出：
  ⚡ 全球風險信號：🟢 平靜 / 🟡 微波 / 🔴 警戒
  📋 關鍵事件列表（最多 5 則，每則含：事件摘要、衝突分數、涉及國家、與台股關聯度）
```

### 模組 B：歷史模式比對引擎 (Pattern Matching Engine)

```
輸入：當前的「情境指紋」
  - VIX 水位區間
  - 殖利率曲線斜率
  - 外資連續買賣超天數
  - GDELT 衝突分數水位
  - Fear & Greed 指數區間
  - 台股距離前高的回檔幅度

處理：
  1. 把當前情境編碼為一組特徵向量
  2. 在歷史資料庫中搜尋最相似的 5 個時期
  3. 統計那 5 個時期「之後 1 週 / 1 月 / 3 月」台股的走勢

輸出：
  🔮 歷史相似度最高的時期：
    1. 2022年10月 Fed 激進升息末期（相似度 87%）→ 之後 3 個月漲 18%
    2. 2020年3月 COVID 恐慌底部（相似度 72%）→ 之後 3 個月漲 35%
    3. 2018年10月 中美貿易戰（相似度 68%）→ 之後 3 個月跌 8%

  📊 歷史勝率統計：
    - 類似情境後 1 週上漲機率：60%（3/5 次）
    - 類似情境後 1 月上漲機率：80%（4/5 次）
    - 平均漲幅：+12.4% / 平均跌幅：-6.2%
```

### 模組 C：明日事件日曆 (Tomorrow's Event Calendar)

```
輸入：
  - 已知排程事件（FOMC 會議、台灣 CPI 公布、法說會、除息日、選舉投票日...）
  - GDELT 趨勢外推（衝突是否持續升溫？）
  - 技術面關鍵價位（明天可能觸碰的支撐/壓力位）

輸出：
  📅 明日關鍵事件：
    09:00 台灣 2 月 CPI 公布 → 若 > 2.5% 可能壓抑科技股
    10:30 台積電法說會 → 上次法說會後 5 日漲 3.2%
    21:30 美國非農就業數據 → 預期 20 萬，若 > 25 萬 → Fed 鷹派 → 台股壓力

  ⚠️ 潛在黑天鵝監控：
    - 中東衝突 GDELT 分數連續 3 天上升（目前 6.2，警戒線 7.0）
    - 日圓兌美元逼近 160 心理關卡（目前 158.7）
```

```

  7. 仲裁官輸出最終「明日預報」
```

## 七、關鍵設計原則

1. **不是預言，是機率**：永遠顯示「歷史上類似情境的勝率」，不說「明天一定漲」
2. **失效條件必須明確**：每個預報都附帶「如果 X 發生，這個判斷就不成立」
3. **資料新鮮度透明**：每筆資料標注取得時間，讓用戶知道資訊的時效性
4. **惡魔代言人不能省**：再樂觀的預報都要有反方質疑
5. **30 秒讀完**：白話版在最上面，想深入的人自己展開面板看
